# Cloud Computing Demonstration Series

This repository contains a series of demonstrations covering cloud computing technologies including AWS, Google Cloud Platform (GCP), containerization, machine learning, microservices, monitoring, and CI/CD automation.

The demonstrations illustrate how modern cloud applications can be deployed, monitored, and automated using industry tools and platforms.

## Technologies Demonstrated

The demonstrations include the following cloud and development technologies:

- Amazon Web Services (AWS)
- Google Cloud Platform (GCP)
- Docker containerization
- Microservice architectures
- Pub/Sub messaging patterns
- Google BigQuery and BigQuery ML
- Prometheus and Grafana monitoring
- CI/CD automation with GitHub Actions

---

## Purpose

These demonstrations illustrate key concepts in cloud-native application development, including containerized services, distributed system communication, machine learning in the cloud, application monitoring, and automated development workflows.
